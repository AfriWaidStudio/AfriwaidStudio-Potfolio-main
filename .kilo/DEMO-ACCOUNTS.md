## Quick Login Guide

1. **Super Admin**: Full system access
2. **Admin**: User/content management
3. **Moderator**: Chat/log moderation
4. **Auditor**: Read-only compliance view
5. **Developer**: System diagnostics
6. **Operator**: Project deployment
7. **Client**: Project tracking & billing
8. **User**: Basic authenticated access
